# Icon

---

<Story id="components-icon--default" />

---

The `Icon` component can be used with two different design systems:

- [Lightning Design System](https://www.lightningdesignsystem.com/icons/)
  - `svg` based icons
  - More icons forth coming in a future update
- [Google Material](https://material.io/resources/icons/?style=baseline)
  - Font based icons

## Quick Start ⚡️

```jsx
import React from 'react';
import { Icon } from '@one-conga/ui-components';

const MyComponent = () => {
  // ...things

  return <Icon icon="search" />;
};
```

# DeleteConfirmationDrawer

---

<Story id="components-deleteconfirmationdrawer--default-from-the-right" />

---

The `DeleteConfirmationDrawer` component allows for an easy, unobtrusive way to double confirm destructive actions with your user. In order to be properly hidden from view, it _must_ be wrapped in a containing element with specific style properties, which can be seen in the "Quick Start" below.

## Quick Start ⚡️

```scss
// MyComponents.scss

.delete-confirmation-container {
  position: relative;
  overflow: hidden; // Can be adjusted for overflow-x/overflow-y based on use case
}
```

```jsx
import React, { useState } from 'react';
import { DeleteConfirmationDrawer } from '@one-conga/ui-components';

import './MyComponent.scss';

const MyComponents = () => {
  const [isShown, toggleIsShown] = useState(false);

  const handleCancel = () => console.log('Cancel');

  const handleConfirm = () => console.log('Confirm');

  return (
    <div className="delete-confirmation-container">
      <DeleteConfirmationDrawer
        isShown={isShown}
        onCancel={handleCancel}
        onConfirm={handleConfirm}
      />
    </div>
  );
};
```

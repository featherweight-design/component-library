# TextArea

---

<Story id="components-form-textarea--types" />

---

The `TextArea` component is a near one-for-one clone of that from Salesforce's Lighting Design System in terms of styling. Small updates have been made to simplify the developer experience, as well as minor enhancements that separate Conga's look from that of LDS (e.g. an initial focus, maximum/minimum character limits and messages, etc.).

To completely fulfill LDS `<textarea>` functionality:

- [ ] `readOnly`

## Quick Start ⚡️

```jsx
import React, { useState } from 'react';
import { TextArea } from '@one-conga/ui-components';

const MyComponent = () => {
  const [value, updateValue] = useState('');

  const handleUpdateValue = ({ target: { value } }) => updateValue(value);

  return <TextArea name="default" value={value} onChange={handleUpdateValue} />;
};
```

# CircleLoader

---

<Story id="components-loaders-circleloader--default" />

---

A simple, animated loader that uses an `<svg>` element under the hood. This allows for custom sizing/coloring without loss of quality or performance.

## Quick Start ⚡️

```jsx
import React from 'react';
import { CircleLoader } from '@one-conga/ui-components';

const MyComponent = () => {
  // ...things

  return <CircleLoader />;
};
```

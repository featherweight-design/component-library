# LoadingSpinner

---

<Story id="components-loaders-loadingspinner--with-no-delay" />

---

The `LoadingSpinner` is a more visually complex and decorative loader initially developed for the AI Analyze application by Roland Arnold. This loader is great for page loading.

## Quick Start ⚡️

```jsx
import React from 'react';
import { LoadingSpinner } from '@one-conga/ui-components';

const MyComponent = () => {
  // ...things

  return <LoadingSpinner />;
};
```

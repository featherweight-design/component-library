# Checkbox

---

<Story id="components-form-checkbox--default" />

---

The `Checkbox` component is a grouping of checkbox inputs ordered in a stacked (`block`) layout. It can also accept a custom "Other" option that allows for a custom input value.

## Quick Start ⚡️

```jsx
import { Checkbox } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  const [options, updateOptions] = useState([
    { checked: true, label: 'Red' },
    { checked: false, label: 'Green' },
    { checked: false, label: 'Blue' },
  ]);

  const handleChange = ({
    target: { name, checked },
  }: ChangeEvent<HTMLInputElement>): void => {
    const newOptions = options.map(option =>
      option.label === name ? { ...option, checked } : option
    );

    updateOptions(newOptions);
  };

  return (
    <div style={{ display: 'flex ' }}>
      <Checkbox
        label="Favorite Primary Color"
        onChange={handleChange}
        options={options}
      />
    </div>
  );
};
```

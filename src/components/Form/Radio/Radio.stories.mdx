# Radio

---

<Story id="components-form-radio--default" />

---

The `Radio` component is a grouping of Radio inputs ordered in a stacked (`block`) layout. It can also accept a custom "Other" option that allows for a custom input value.

## Quick Start ⚡️

```jsx
import { Radio } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  const [selected, handleChangeSelected] = useState('');

  return (
    <div>
      <Radio
        label="Baby Names"
        selected={selected}
        options={['Torin', 'Juniper', 'Pooch', 'Luna']}
        onChange={(event): void => {
          handleChangeSelected(event.target.name);
        }}
      />
    </div>
  );
};
```

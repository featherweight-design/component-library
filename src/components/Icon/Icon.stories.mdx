The `Icon` component can be uses [Google Material](https://material.io/resources/icons/?style=baseline), which are font based icons.

## Quick Start ⚡️

```jsx
import React from 'react';
import { Icon } from '@f-design/component-library';

const MyComponent = () => {
  // ...things

  return <Icon icon="search" />;
};
```

# ActionButton

---

<Story id="components-buttons-actionbutton--default" />

---

The `ActionButton` component is a more mobile friendly button with a larger footprint than the `Button` component. Its primary use case is for specific call to actions (e.g. "complete", "cancel", "add", etc.).

## Quick Start ⚡️

```jsx
import React from 'react';
import { ActionButton } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  const handleOnClick = () => console.log('Click!');

  return <ActionButton icon="check" onClick={handleOnClick} />;
};
```

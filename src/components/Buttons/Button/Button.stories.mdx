The `Button` component is a basic button compon.

## Quick Start ⚡️

```jsx
import React from 'react';
import { Button } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  const handleOnClick = () => console.log('Click!');

  return <Button icon="check" onClick={handleOnClick} />;
};
```

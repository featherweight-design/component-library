The `Button` component is a basic button component.

## Quick Start ⚡️

```jsx
import { Button } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  const handleOnClick = () => console.log('Click!');

  return (
    <Button variant="brand" onClick={handleOnClick}>
      Click me!
    </Button>
  );
};
```

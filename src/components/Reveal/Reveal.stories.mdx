# Reveal

---

<Story id="components-reveal--default" />

---

The `Reveal` component is a lightweight solution to obfuscate elements from users that need to remain within the normal document flow. Any size of element can be hidden by the `Reveal` component, though the transition styling works best with smaller elements such as buttons or inputs.

## Quick Start ⚡️

```jsx
import React from 'react';
import { Button, Reveal } from '@one-conga/ui-components';

const MyComponent = () => {
  const [isShown, toggleReveal] = useState(false);

  return (
    <div>
      <Button type="brand" onClick={() => toggleReveal(!isShown)}>
        Toggle reveal
      </Button>

      <Reveal isShown={isShown}>
        <Button type="destructive">Secret Destroy Button</Button>
      </Reveal>
    </div>
  );
};
```

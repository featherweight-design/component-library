# CircleLoader

---

<Story id="components-loaders-circleloader--default" />

---

The `CircleLoader` componet is a basic, pure CSS loader that can take custom colors and sizes. It is imbedded in several other components as part of a "loading" state.

## Quick Start ⚡️

```jsx
import { CircleLoader } from '@f-design/component-library';

const MyComponent = () => {
  // ...things
  return <CircleLoader />;
};
```
